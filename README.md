## 機能

コマンド ```/where <MCID>``` を使用することで、対象のプレイヤーに以下の効果を付与します。

- 発光の効果を付与
- 位置情報をチャットに表示
- Xaero's Mapでプレイヤーの位置のウェイポイントを共有(※導入しないと使用できません)
